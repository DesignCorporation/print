## Storage (Cloudflare R2)
Добавьте ключи для загрузки макетов:
- `R2_ACCOUNT_ID`
- `R2_ACCESS_KEY_ID`
- `R2_SECRET_ACCESS_KEY`
- `R2_BUCKET`
- `R2_PUBLIC_URL` (опционально, например `https://<bucket>.<account>.r2.cloudflarestorage.com`)
